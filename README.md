# 📌 System Overview

A fully branded, production-ready recruitment onboarding system built for Kyakabi Group Limited.

This system is:

- 🔐 Authentication-first (No landing page)
- 📱 Mobile-first & Fully Responsive
- ⚡ PWA Enabled (Installable + Offline Support)
- 💾 Local Storage Based (No Cloud Backend)
- 🎨 Fully Branded (#009966 primary, #f05922 secondary)
- 🧑‍💼 Includes Admin Panel
- 📩 Integrated Email & WhatsApp Notification Flow (UI-ready)

# 🚀 Local Development Setup

```bash
# Clone repository
git clone https://github.com/Hatalabdallah/kyakabi-Job-onboard.git

# Navigate into project
cd kyakabi-Job-onboard

# Install dependencies
npm install

# Start development server
npm run dev
````

Application runs on:

```
http://localhost:8080
```
